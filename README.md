
# face-tracker
Python utility to track and denoise face
This tool can track and denoise face with your webcam in realtime.

## HOW TO USE:
>     1. Clone the repo
>     2. Call FaceTracker.py from python interpreter
>     3. Tada ! you are good to go !
>
